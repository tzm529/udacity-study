# 统计学：决策的科学项目说明

### 背景信息

在一个 Stroop （斯特鲁普）任务中，参与者得到了一列文字，每个文字都用一种油墨颜色展示。参与者的任务是将文字的打印颜色大声说出来。这项任务有两个条件：一致文字条件，和不一致文字条件。在一致文字条件中，显示的文字是与它们的打印颜色匹配的颜色词，如“红色”、“蓝色”。在不一致文字条件中，显示的文字是与它们的打印颜色不匹配的颜色词，如“紫色”、“橙色”。在每个情况中，我们将计量说出同等大小的列表中的墨色名称的时间。每位参与者必须全部完成并记录每种条件下使用的时间。

```
假设显示的文字是与打印颜色匹配词不一致时，会影响说出打印墨色的时间。

为了证实以上结论，本案例可以使用配对样本双尾T检验来验证。

零假设：显示的文字与打印颜色匹配词不一致时，不会影响说出打印墨色的时间，即一致文字条件时的均值等于不一致文字条件时的均值。

对立假设：显示的文字与打印颜色匹配词不一致时，影响说出打印墨色的时间，即一致文字条件时的均值不等于不一致文字条件时的均值。

支持本案例使用配对样本双尾T验证的理由如下：
1. 并不知道总体相关的参数，如平均值和方差等，我们手中仅有少量样本统计数据，又因为这里的试验是同一类人分别试验两种不同的方案，因此可以使用配对样本的双尾T统计来处理.

2. 此试验来自同一独立受试者两次不同测试,总体是一个独立的。

3. 两个样本估算出来的总体方差差距并不大，我们可近似认为是相等的。

4. 执行Stroop时，当尝试的次数越多，时间分布越接近于正态分布，因此我们可以假设两个样本的总体分布均接近于正态分布。

```

*这里先定义一些变量：*

一致文字条件样本，定义为a

不一致文字条件样本，定义为b

`$H_0$`：表示零假设

`$H_A$`：表示对立假设

`$u_a$`表示一致文字条件总体的均值

`$u_b$`表示不一致文字条件总体的均值

*具体假设的表达式如下：*
```math
H_0: u_a = u_b

H_A: u_a != u_b
```
